Class: Manager

Manager(mdTarget)

This class manages our app data. Call its functions to change data. Add / get returns proxied notes, which automatically save changes to localStorage. If you modify unproxied data, make sure to call save yourself. If you modify a sub property, make sure to save yourself. We store all the notes in an object, keyed by note UUID. Similarly for projects.
